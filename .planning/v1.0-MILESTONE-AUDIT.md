## Remaining Integration Note (Low Severity)

**Hotkey live reload not implemented** — When a user changes hotkey bindings in Settings, the new values are persisted to UserDefaults but the running HotkeyMonitor still uses the original values. The new bindings take effect on next app launch. This is acceptable for v1.0 — most macOS apps with global hotkeys behave similarly.
