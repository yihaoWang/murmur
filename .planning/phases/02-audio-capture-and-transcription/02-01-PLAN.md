2. Create Typeness/Core/AudioCaptureEngine.swift as a Swift `actor`:
   - `enum AudioError: Error { case converterCreationFailed, microphonePermissionDenied, engineStartFailed }`
   - Private properties: `engine: AVAudioEngine`, `samples: [Float]`, `converter: AVAudioConverter?`
   - Private constant: `targetFormat = AVAudioFormat(commonFormat: .pcmFormatFloat32, sampleRate: 16_000, channels: 1, interleaved: false)!`
   - Max sample cap: `private let maxFrames = 480_000` (30 seconds at 16kHz)
   - `func checkMicrophonePermission() async -> Bool` — uses `AVCaptureDevice.authorizationStatus(for: .audio)`, returns true if `.authorized`, calls `AVCaptureDevice.requestAccess(for: .audio)` if `.notDetermined`, returns false otherwise
   - `func start() throws` — gets `engine.inputNode.outputFormat(forBus: 0)` as inputFormat, creates `AVAudioConverter(from: inputFormat, to: targetFormat)`, clears samples array, installs tap with `bufferSize: AVAudioFrameCount(inputFormat.sampleRate * 0.1)` at `inputFormat`, calls `engine.start()`. The tap callback dispatches `Task { await self?.process(buffer: buffer) }`
   - `private func process(buffer: AVAudioPCMBuffer)` — uses the "filled" boolean pattern from research: compute `frameCount = AVAudioFrameCount(Double(buffer.frameLength) * (16_000.0 / buffer.format.sampleRate))`, create output PCMBuffer with `frameCapacity: frameCount + 16`, call `converter.convert(to:error:inputBlock:)` with `var filled = false` guard, append `outBuffer.floatChannelData![0]` samples. Cap total samples at `maxFrames`.
   - `func stop() -> [Float]` — calls `engine.inputNode.removeTap(onBus: 0)`, `engine.stop()`, returns samples array
   - `import AVFoundation` at top

3. Add `"Core/AudioCaptureEngine.swift"` to Package.swift sources array after `"Core/ModelManager.swift"`.
